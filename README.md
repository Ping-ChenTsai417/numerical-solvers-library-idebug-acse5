# Linear solvers

#### Solvers for *dense* matrix implemented:

1.  Jacobi method
2.  Gauss-Seidel method
3.  Gaussian method
4.  LU decomposition
5.  Gauss-Jordan inverse method
6.  Cholesky method
7.  Conjugate gradient method

#### Solvers for *sparse* matrix implemented:

1.  Jacobi method
2.  Gauss-Seidel method
3.  Conjugate gradient method
4.  Cholseky method
